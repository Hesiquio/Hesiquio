# ¡Hola! Soy Hesiquio Zárate 👨‍🏫💻
### Desarrollador de Software | Docente Investigador | Impulsor de AI-DLC
Soy un apasionado de la tecnología con un enfoque híbrido entre la ingeniería de software de alto rendimiento y la educación superior. Mi misión es transformar el aula y la gestión institucional mediante herramientas disruptivas, priorizando la arquitectura local-first y la experiencia del usuario.
---
### 🚀 Proyectos Destacados
*   **[SaidVerso](https://github.com/Hesiquio/saidverso)**: Plataforma educativa gamificada multiplataforma (Web/Móvil) diseñada para el aprendizaje dinámico mediante retos interactivos.
*   **Archivística AI-DLC (SIAI)**: Sistema de gestión documental académica con arquitectura **Local-First**, construido sobre **Bun**, **Hono** y **PGlite** para un rendimiento excepcional y disponibilidad offline.
*   **[Classroom Clash](https://github.com/Hesiquio/classroomclash)**: Ecosistema de gamificación para fomentar la competitividad lógica en el aula.
*   **Trayectorias Académicas**: Dashboard de seguimiento estudiantil integrado con **Angular** y **Supabase**.
